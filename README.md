# Tienda Online - Proyecto Final Web

Este es el proyecto final para la clase de Programación Web. Consiste en una tienda en línea funcional con un catálogo de 30 productos.

## Tecnologías Utilizadas

*   **Frontend:** HTML, CSS, JavaScript
*   **Backend:** Node.js, Express
*   **Base de Datos:** MySQL (SQL)

## Características

*   Catálogo de 30 productos
*   Carrito de compras
*   Sistema de usuarios y autenticación
*   Panel de administración
*   Base de datos relacional

## Instalación

1.  Clonar el repositorio.
2.  Instalar dependencias: `npm install`
3.  Configurar la base de datos (ver `INSTRUCCIONES-SETUP.md`).
4.  Iniciar el servidor: `npm start`
